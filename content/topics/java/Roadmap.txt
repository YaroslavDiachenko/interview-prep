For a **Senior Java Developer** interview, interviewers usually expect more than knowing the language syntax. They want to see that you understand **how Java works internally**, can reason about performance, concurrency, memory, and design trade-offs.

Here's a roadmap organized in the order I'd recommend studying.


# Topic: Java Language Fundamentals

		### Variables & Types

		* Primitive types
		* Wrapper classes
		* Autoboxing/unboxing
		* Type conversion
		* Overflow
		* String vs StringBuilder vs StringBuffer

		### Operators

		* == vs equals()
		* instanceof
		* Pattern matching for instanceof (Java 17+)

		### Control Flow

		* switch expressions
		* enhanced switch
		* loops
		* labels (rare)
